🎁 Gutscheine

Einmalige Einrichtung — danach läuft alles automatisch auf allen Geräten.

Aus Google Cloud Console → APIs & Dienste → Anmeldedaten
Aus der URL Ihres Google Sheets: docs.google.com/spreadsheets/d/ID/edit
Nur zum endgültigen Löschen aus dem Archiv nötig

Einrichtungsanleitung

1
Öffnen Sie console.cloud.google.com → wählen Sie Ihr Projekt „Gutscheine"
2
„APIs & Dienste" → „Bibliothek" → suchen Sie nach Google Sheets API → aktivieren
3
„APIs & Dienste" → „Anmeldedaten" → Ihren API-Schlüssel kopieren (beginnt mit AIzaSy)
4
Ihre OAuth Client-ID kopieren (endet auf .apps.googleusercontent.com)
5
Gehen Sie zu sheets.google.com → erstellen Sie eine neue leere Tabelle → benennen Sie sie „Gutscheine"
6
Kopieren Sie die Spreadsheet-ID aus der URL (der lange Code zwischen /d/ und /edit)
7
Teilen Sie die Tabelle mit der E-Mail Ihrer Frau (Bearbeiter-Zugriff)
8
Füllen Sie die Felder oben aus und klicken Sie „Einrichtung abschließen" — Google fragt nach Berechtigung → Zulassen
