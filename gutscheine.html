Einrichtungsanleitung

  1. Öffnen Sie console.cloud.google.com und erstellen Sie ein neues Projekt (z.B. „Gutscheine")
  2. Aktivieren Sie die Google Drive API unter „APIs & Dienste" → „Bibliothek"
  3. Erstellen Sie unter „APIs & Dienste" → „Anmeldedaten" einen API-Schlüssel und kopieren Sie ihn oben ein
  4. Erstellen Sie eine OAuth 2.0 Client-ID (Typ: Webanwendung). Fügen Sie als autorisierte JavaScript-Quelle die URL dieser Datei hinzu (oder https://storage.googleapis.com wenn Sie Google Drive direkt nutzen)
  5. Erstellen Sie in Ihrem Google Drive eine leere Datei namens gutscheine.json mit dem Inhalt [] und teilen Sie diese Datei mit Ihrer Frau
  6. Öffnen Sie die JSON-Datei in Drive, kopieren Sie die Datei-ID aus der URL und fügen Sie sie oben ein
  7. Wiederholen Sie diese Einrichtung auf dem zweiten Gerät Ihrer Frau — verwenden Sie dieselbe Datei-ID und dasselbe Passwort
